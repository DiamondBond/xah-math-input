An emacs minor mode for inputing math symbols and Unicode symbols.

[[http://melpa.org/#/xah-math-input][file:http://melpa.org/packages/xah-math-input-badge.svg]]

Home Page: http://ergoemacs.org/emacs/xmsi-math-symbols-input.html

Using straight.el:
#+begin_src emacs-lisp
  (use-package xah-math-input
	:straight
	( :repo "diamondbond/xah-math-input"
	  :host github
	  :type git
	  :files ("*")))
#+end_src
